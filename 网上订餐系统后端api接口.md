# 网上订餐系统后端api接口

## 

## 1. 用户登录api

请求方式: post

地址：

传递参数

| 参数     | 说明       | 备注 |
| :------- | ---------- | ---- |
| username | 用户账号   | 必传 |
| pwd      | 输入的密码 | 必传 |

返回参数

| 参数    | 说明                  | 备注 |
| ------- | --------------------- | ---- |
| message | 成功/失败             |      |
| code    | 状态吗200成功/500失败 |      |

## 2. 用户注册api

请求方式：post

地址：

传递参数

| 参数     | 说明       | 备注 |
| -------- | ---------- | ---- |
| username | 用户账号   | 必传 |
| pwd      | 输入的密码 | 必传 |
| age      | 年龄       | 必传 |
| sex      | 性别 0/1   | 必传 |
| email    | 邮箱       | 必传 |
| tel      | 电话       | 必传 |

返回参数：

| 参数    | 说明                  | 备注 |
| ------- | --------------------- | ---- |
| message | 成功/失败             |      |
| code    | 状态吗200成功/500失败 |      |
